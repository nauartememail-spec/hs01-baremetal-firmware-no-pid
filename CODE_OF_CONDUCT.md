# Нормы поведения (Code of Conduct)

Помните главное правило общения в нашей разработке:

> **«Ребята, давайте жить дружно!» — Кот Леопольд.**

### У нас запрещено:
* Токсичность.
* Оскорбления.
* Хейт в комментариях к коду.
* Разведение споров не по делу (флуд).
* Угрозы.

Давайте уважать друг друга. Не забывайте, по ту сторону экрана тоже сидит человек!
