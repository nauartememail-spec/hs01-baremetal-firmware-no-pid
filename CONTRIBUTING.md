# Руководство по участию в проекте (Contributing Guide)

Приветствую! Если вы хотите помочь в разработке кастомной параболической прошивки для Fnirsi HS-01, я буду рад вашему участию.

## Как можно помочь:
* **Тестирование математики:** Если у вас есть возможность проверить алгоритм перевёрнутой параболы в симуляторах или на аналогичном железе — делитесь результатами.
* **Разработка Bare-Metal:** Проекту нужен чистый C-код для работы с периферией микроконтроллера CH571F (дисплей OLED (нескольких типов) , кнопки, АЦП, таймеры ШИМ, загрузчик)
* **Оптимизация формул:** Приветствуются идеи по улучшению целочисленных (int32_t) вычислений без использования плавающей точки. Желательно делать вообще без float, насколько это возможно. Всё потому, что в будущем мы будем разгонять прошивку на максимум, и важно, чтобы математические операции выполнялись в один такт.

## Как предложить изменения:
1. Сделайте форк (Fork) этого репозитория.
2. Создайте свою ветку для улучшений (`git checkout -b feature/cool-feature`).
3. Внесите изменения и отправьте Pull Request (PR) в ветку `main`.
